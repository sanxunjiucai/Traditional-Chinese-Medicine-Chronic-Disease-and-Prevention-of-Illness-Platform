## 关键设计决策

1. **渐进式重构** - v1/v2 并存，不影响现有系统
2. **证据链优先** - 所有判断必须有证据支撑
3. **确认机制** - 所有写操作必须医生确认
4. **工具化封装** - 业务能力封装为可被 AI 调用的工具
5. **最小化实现** - 每个阶段只做核心功能
