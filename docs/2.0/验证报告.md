## 已知问题

### 1. 服务器启动问题
**问题**: 表重复定义导致服务器无法启动
**原因**: 旧的 plan_distributions 表与新表冲突
**解决方案**: 需要清理旧表或使用 extend_existing=True

### 2. 编码问题
**问题**: Windows 控制台 GBK 编码
**解决方案**: 已在测试脚本中移除特殊字符
